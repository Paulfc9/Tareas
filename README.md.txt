Para resolver la tarea de la unidad 2 :

Se debe de declarar una variable en este caso nuestra variable se
llamara mes.

con el comando prompt nos habre una ventanita en la que el usuario
nos debe ingresar un numero que se guardara en nuestra variable
mes.

Con el uso del condicional if podremos verificar si nuestra varia-
ble mes cumple los requisitos para ingresar al switch.

si mes >= 1 y mes <= 12 podra ingresar al switch.

Dentro de nuestro switch vamos a tener 12 casos es decir 1 caso por mes
por ejemplo:

en caso que el usuario ingrese mes 2, la condicion se cumple y entra al
swtich y en el caso numero 2 tenemos que la estacion es "invierno".

De lo contrario nos dira que el valor ingresado es invalido.
